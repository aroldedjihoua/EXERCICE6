#  Exercice Java - Classe Personne

##  Description
Cet exercice montre la différence entre une variable `static` et une variable d’instance en Java.

##  Objectif
- Créer une classe `Personne`
- Utiliser un compteur global (static)
- Utiliser un compteur individuel (non static)
- Comprendre le fonctionnement des objets

##  Fonctionnement
À chaque création d’un objet :
- `nbInstances` augmente (total des objets)
- `nbLocal` augmente (propre à chaque objet)

##  Exécution
```bash
javac *.java
java Main
``` id="run1"
